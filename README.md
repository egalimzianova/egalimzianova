## Портфолио
### Проекты под НДА

#### Логистическое приложения для газовой компании
Роль: System Analyst/Project coordinator
Команда: 15-25 человек
Архитектура: микросервисная
Задача: Разработка логистического приложения для газовой компании в рамках импортзамещения и подготовка миграции данных. 
Моя зона ответственности:
- Создание структуры беклога
- Контроль наполнения беклога, поддержание в актуальном состоянии, мониторинг покрытия требованиями
- Написание User story и создание мокапов
- Создание дешбордов проекта для анализа эффективности команды разработки и прогнозирования сроков завершения (WIP, Cycle time, Lead time, Cumulative flow diagram, Throughput )

#### Проект: POS-решение японского заказчика
Роль: System Analyst/Project coordinator
Команда: 15 человек
Архитектура: монолит
Задача:
Разработка корневой логики кроссплатформенного POS-решения на C# для японского заказчика.

Моя зона ответственности:
- Выявление, перевод и уточнение требований для передачи в разработку
- Координация команды, распределение задач, отслеживание прогресса
